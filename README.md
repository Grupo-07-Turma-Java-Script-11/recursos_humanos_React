# 👥 People Flow — Sistema de Gestão de Recursos Humanos

> Projeto Integrador — Grupo 07 | Turma JavaScript 11 | Generation Brasil

---

## 📋 Descrição

O **People Flow** é uma aplicação fullstack desenvolvida como Projeto Integrador da [Generation Brasil](https://brazil.generation.org/), com o objetivo de simular um sistema de gestão de Recursos Humanos. A plataforma apresenta institucionalmente a missão, visão e valores da organização, além de estruturar de forma visual e intuitiva as informações estratégicas do setor de RH.

A aplicação foi construída com **React**, **TypeScript** e **Vite**, seguindo boas práticas de componentização, estilização e deploy em produção.

🔗 **Acesse o projeto em produção:** [recursos-humanos-react.vercel.app](https://recursos-humanos-react.vercel.app)

## ⚙️ Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- [npm](https://www.npmjs.com/) (geralmente incluído com o Node.js)
- [Git](https://git-scm.com/)

---

## 🔧 Instalação e Configuração

### 1. Clone o repositório

```bash
git clone https://github.com/Grupo-07-Turma-Java-Script-11/recursos_humanos_React.git
```

### 2. Acesse o diretório do projeto

```bash
cd recursos_humanos_React
```

### 3. Instale as dependências

```bash
npm install
```

---

## ▶️ Executando a Aplicação

```bash
# Modo desenvolvimento
npm run dev
```

A aplicação estará disponível em: **`http://localhost:5173`**

```bash
# Build para produção
npm run build

# Pré-visualizar build de produção
npm run preview
```
